# cubist-hackathon-team-point5
Cubist Hackathon Chess Engine Project 2026


Our chess engine references the open source model from Stockfish. We will be having different “modes”. You will be able choose to play against different GrandMasters, such as Magnus Carlsen. Each of the 4 grandmaster options represents different playing styles.
| Playing Style | Grandmasters | Description |
|---|---|---|
| Aggresive Player | Garry Kasparov | Aggresive initiative plays |
| Strategic Player | Tigran Petrosian | Deep thinking to prevent counterplays |
| Defensive Player | Anatoly Karpov | Prioritize neutralizing opponents' attacks |
| Universal Player | Magnus Carlsen | Maximizes winning chances by exploiting small inaccuracies |

We’re taking the historical data of each GM to learn about how they play. We change the parameters of the chess engine to mirror their strategy, whether they’re more defensive, an attacker, positional, or universal
